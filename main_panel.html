Privacy Policy

Effective: 11.07.2025

This Privacy Policy applies to the use of this application by authorised migration agents in Australia. The application is a tool designed to support the management and analysis of application data submitted by clients (applicants) for the purpose of migration assistance.

1. Data Controller and Processor Roles

The application functions as a data processor, acting on behalf of authorised migration agents, who act as the data controllers under the Privacy Act 1988 (Cth). Migration agents are responsible for ensuring that any personal or sensitive information uploaded into the system is collected and used in accordance with the Australian Privacy Principles (APPs).

2. Purpose of Data Processing

The system processes personal and sensitive information provided by or on behalf of applicants, including but not limited to:

  • Identification details
  • Contact information
  • Employment and education history
  • Supporting documentation relevant to migration applications

Data is processed exclusively for migration-related services as instructed by the migration agent.

3. Access Restrictions

Only authorised migration agents with valid credentials are permitted to access the application. Applicants do not have direct access to the system and do not interact with it directly.

4. Data Retention and Deletion

To protect applicant privacy and minimise data risk, all uploaded files and applicant data are automatically deleted from the system 30 days after upload. Migration agents are responsible for storing required documentation outside the platform if long-term retention is needed.

5. Security Measures

The application implements industry-standard security practices, including:

  • Encryption in transit and at rest
  • User authentication and access control
  • Regular security updates and vulnerability assessments

All data is hosted in secure infrastructure located in compliance with Australian data sovereignty requirements.

6. Migration Agent Responsibility

It is the responsibility of the migration agent to:

  • Obtain appropriate consent from applicants where required
  • Ensure that applicants are informed about the processing of their data
  • Comply with all obligations under the Privacy Act 1988 and relevant regulations
